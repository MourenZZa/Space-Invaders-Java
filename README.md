## SPACE INVADERS
The project was built as a learning exercise to practice Java fundamentals and get comfortable with the language through a hands-on, end-to-end application. In other words: it’s not “enterprise-grade” code pretending to be perfect—it’s code written to learn by doing, experimenting, iterating, and improving along the way.
